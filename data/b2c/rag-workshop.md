# RAG Workshop — практикум по retrieval-augmented generation

**Аудитория:** B2C, разработчики с базовым опытом LLM.

## О курсе

Интенсив **rag-workshop** — двухнедельный практикум по построению RAG-систем:
chunking, embeddings, vector stores (Chroma, Pinecone), evaluation.

## Формат

- Длительность: 2 недели
- Формат: live-coding сессии + проект
- Уровень: средний

## Результат

Собственный RAG-пайплайн с индексацией PDF/MD и semantic search.
