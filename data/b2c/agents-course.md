# Курс Agents — разработка AI-агентов

**Аудитория:** B2C, индивидуальные разработчики и энтузиасты.

## О курсе

Курс **agents** — флагманский продукт LLMStart для тех, кто хочет научиться создавать
production-grade AI-агентов на Python. Программа охватывает LangChain, ReAct-паттерн,
tools, RAG и observability.

## Формат

- Длительность: 8 недель
- Формат: онлайн, практические домашние задания
- Уровень: средний (нужен базовый Python)

## Что вы освоите

- Архитектуру ReAct-агентов с tool calling
- RAG по корпоративной базе знаний
- Интеграцию Langfuse для traces
- Деплой агента как FastAPI-сервиса
